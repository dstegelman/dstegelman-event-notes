Django Class Based Views¶

Presenter: Nick Lang

Description:

Class Based Views

Background¶

  • Developer at Lab 305
  • Worked at Journal World

Overview¶

  • Class version of generic function views.
  • Base, Dates, Detail, Edit, List
  • CRUD Views
  • Subclassed (Mixins)

Mixins¶

Object with methods that can be overridden to customize behavior.

  • ContentMixin
  • TemplateResponseMixin (Adds context variables)
  • Date Views, Many.

Detail Views¶

  • Single object
  • Primary key or slug

Mixins¶

  • SingleObjectMixin
  • SingleObjectTemplateResponsemixin

Crud Views¶

  • Create/Read/Update/Delete

Mixins¶

  • MomdelFormMixin

Base View¶

Why¶

  • Elegance

  • Lots of possibilities for mixins.
  • Django Braces
